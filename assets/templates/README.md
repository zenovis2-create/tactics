# Asset Spec Templates

This folder contains production templates for game-art asset work.

Use these templates when starting a new asset:

- `character/spec.md`
- `sprite_character/spec.md`
- `prop/spec.md`
- `tile/spec.md`
- `map/spec.md`

Workflow:

1. Copy the closest template into the target asset folder.
2. Fill in only what is known.
3. Keep unknown items explicit instead of guessing silently.
4. Use the completed spec as the source of truth for Rhino, Krita, Godot, and AI-image tasks.
