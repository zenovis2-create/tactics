# Sprite Anchor Enemy Skirmisher Prompt Pack V01

## Purpose

This prompt pack generates the first production-facing concept sheets for `Sprite Anchor Enemy Skirmisher`.
The goal is to create a second hostile lane that expands enemy class distance without breaking the shared style family.

Use this with:

- `/Volumes/AI/tactics/assets/characters/sprite_anchor_enemy_skirmisher/spec.md`
- `/Volumes/AI/tactics/docs/character_sprite_pipeline.md`
- `/Volumes/AI/tactics/docs/character_class_silhouette_color_matrix_v01.md`
- `/Volumes/AI/tactics/docs/production/character_visual_identity_pack.md`

## Shared Rules

Apply these rules to every prompt in this pack:

- classic tactical JRPG sprite-sheet feel
- chibi-adjacent tactical proportion
- clean line discipline
- restrained cel-style shading
- gameplay-scale readability first
- transparent or plain neutral background
- no text, no numbering, no speech bubbles
- no cinematic composition
- no western gritty realism
- no giant FX

```text
Primary request: idle animation sheet for a hostile empire skirmisher, designed for a classic tactical JRPG battle sprite pipeline
Subject: compact hostile light skirmisher with slightly enlarged head, lean tactical body, light ranged weapon, compressed enemy posture, restrained ember-red hostile accent, dark authority cloth and leather
Style/medium: classic tactical JRPG sprite-sheet concept art, clean line discipline, restrained cel-style shading, same rendering family as allied sprites and enemy raider
Composition/framing: full-body gameplay sprite sheet on a plain neutral background, front-facing battle orientation, consistent frame-to-frame body scale, 8-frame idle loop concept
Lighting/mood: soft neutral production lighting
Color palette: iron gray, soot leather, dark authority cloth, controlled ember-red accent
Constraints: the unit must read as hostile before color; it must look lighter and faster than Enemy Raider; it must not look like an ally ranger recolor
Avoid: ally softness, magical effects, gritty realism, assassin glamour, giant cloaks, text, panel numbers, background scenery
```
