# Enemy Skirmisher 8Dir Production Brief V01

## Goal

Produce the first eight-direction gameplay base for Enemy Skirmisher.

Required views:

- `front`
- `front_right`
- `right`
- `back_right`
- `back`
- `back_left`
- `left`
- `front_left`


## Read Goal

Enemy Skirmisher must still read as:

- hostile agile threat first
- pursuit hunter second

The main risk is collapse into:

- Tia recolor or generic rogue

## Must Keep Across All Directions

- lean hostile frame
- compressed aggressive stance
- light ranged threat

- front: clearest primary class read
