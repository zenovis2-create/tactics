# Character Anchor Knight Krita Cleanup Guide V01

## Purpose

This guide converts the current anchor knight source sheet into a cleanup-ready reference sheet for review and Rhino blockout support.

Source candidate:

- `/Volumes/AI/tactics/assets/characters/character_anchor_knight/source/character_anchor_knight_sheet_source_v01.png`

Use together with:

- `/Volumes/AI/tactics/assets/characters/character_anchor_knight/spec.md`
- `/Volumes/AI/tactics/docs/style_bible.md`
- `/Volumes/AI/tactics/docs/clean_pass_production_guide_v01.md`

## Cleanup Goal

Produce a clean anchor sheet that:

- preserves the `front`, `side`, and `3/4` turnaround relationship
- removes concept-board clutter and generation residue
- keeps the knight readable as the baseline human-scale asset
- strengthens shield, armor-mass, and navy-accent separation
- is safe to use as Rhino blockout reference

## Step 1. Preserve The Turnaround Structure

- keep the three required views only
- keep the stance neutral and modeling-friendly
- do not crop one view differently enough that proportions become hard to compare

The sheet is reference-first, not poster-first.

## Step 2. Normalize The Background

- remove any poster or concept-board residue
- keep a plain neutral studio background if full transparency hurts sheet readability
- do not leave decorative gradients, labels, or mock presentation borders

If the background remains visible, it must stay plain and non-distracting.

## Step 3. Clean The Silhouette

Prioritize:

- broad shoulder mass
- large shield read
- compact straight sword profile
- heavy lower armor

Remove or reduce:

- tiny trim
- accidental dangling fragments
- extra cloth noise
- soft edge haze that weakens contour readability

## Step 4. Clarify Material Zones

- armor shell should read first
- leather support should read second
- navy cloth accent should stay clear but restrained
- one shield focal mark is enough

Do not let accents become louder than the structure.

## Step 5. Protect The Anchor Read

The cleaned result must still read in one glance as:

1. knight
2. defensive frontline unit
3. worn veteran

If cleanup makes the character more glamorous or more ornate, reverse it.

## Step 6. Export Clean Sheet

Export:

- `character_anchor_knight_sheet_clean_v01.png`

Store it under:

- `/Volumes/AI/tactics/assets/characters/character_anchor_knight/clean/`

## Step 7. Future Runtime Preparation

This lane is currently a reference lane, not a promoted runtime sprite lane.

Later derivatives may include:

- isolated shield reference
- isolated sword reference
- reduced-size modeling board

Do not make those now unless explicitly requested.

## Acceptance Checklist

- three-view turnaround remains intact
- shield is still a dominant silhouette hook
- navy accent is visible but controlled
- no concept-board clutter remains
- sheet supports Rhino blockout without relying on micro-detail
