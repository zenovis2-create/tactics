# Tia 8Dir Production Brief V01

## Goal

Produce the first eight-direction gameplay base for Tia.

Required views:

- `front`
- `front_right`
- `right`
- `back_right`
- `back`
- `back_left`
- `left`
- `front_left`


## Read Goal

Tia must still read as:

- ranged hunter first
- forest skirmisher second

The main risk is collapse into:

- generic fantasy archer or rogue-like dagger user

## Must Keep Across All Directions

- bow curve
- asymmetrical hood or shoulder break
- lean stance

## Quality Bar

- front: clearest primary class read
- diagonals: true in-between orientation, not duplicated side views
- back: the class silhouette still reads from rear mass and posture
- left/right: bow and asymmetry stay visible in profile
- all eight: identity stays stable without relying on FX or background help
