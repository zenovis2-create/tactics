# Sprite Anchor Tia 8Dir Prompt Pack V01

## Purpose

This prompt pack generates the first production-facing eight-direction base set for `Sprite Anchor Tia`.
The goal is to lock directional silhouette and class readability before any animation work.

Use this with:

- `/Volumes/AI/tactics/assets/characters/sprite_anchor_tia/spec.md`
- `/Volumes/AI/tactics/assets/characters/sprite_anchor_tia/8dir_production_brief_v01.md`
- `/Volumes/AI/tactics/assets/characters/sprite_anchor_tia/prompt_pack_v01.md`
- `/Volumes/AI/tactics/docs/style_bible.md`

## Shared Rules

Apply these rules to every prompt in this pack:

- classic tactical JRPG sprite-style character base
- single-direction concept only, not an animation sheet
- chibi-adjacent tactical proportion
- clean line discipline
- restrained cel-style shading
- gameplay-scale readability first
- transparent or plain neutral background
- no text, no numbering, no labels
- no cinematic composition
- no glossy modern anime finish
- no oversized FX or battle effects
- preserve turn-appropriate silhouette in every orientation

## Front View Prompt

```text
Use case: stylized-concept
Asset type: tactical RPG eight-direction character base
Primary request: front directional base for Tia, designed for a classic tactical JRPG battle sprite pipeline
Subject: same character with the clearest front-facing class read
Style/medium: classic tactical JRPG character base art, clean line discipline, restrained cel-style shading, readable gameplay-scale silhouette
Composition/framing: full-body single-view directional base on a plain neutral or transparent background, centered figure, neutral stance, no action pose
Lighting/mood: soft neutral lighting
Constraints: preserve ranged hunter first and forest skirmisher second; keep bow curve, asymmetrical hood or shoulder break, and lean stance readable; do not drift toward rogue or dagger-user posture
Avoid: text, numbering, animation-sheet layout, background scenery, extra characters, giant FX, duplicated views
```
