# Tia Layered 8Dir Production Brief V01

## Goal

Migrate Tia from monolithic 8-direction prep into a layered 8-direction ranged
hunter lane.

## Required Layers

- `base_body`
- `base_outfit`
- `weapon_overlay`

Optional in the lane:

- `upper_armor_overlay`
- `shield_overlay`

## Direction Set

- `front`
- `front_right`
- `right`
- `back_right`
- `back`
- `back_left`
- `left`
- `front_left`

## Read Goal

Tia must read as:

- ranged hunter first
- forest skirmisher second

The main risks are:

- drifting into rogue or dagger-user posture
- letting swappable upper gear erase the lane's asymmetry
- making the bow feel optional

## Layer Guidance

### `base_body`

Keep:

- head
- face
- hair
- body mass
- limb length and light hunter stance

Must not carry:

- bow
- quiver-shaped gear mass that should be swappable
- upper armor that reads as equipable gear

### `base_outfit`

Keep:

- baseline hunter clothing
- lower-body cloth and boot silhouette
- stable asymmetry that defines Tia's forest-ranged identity

Must preserve:

- lighter ranged silhouette than Rian
- more agile stance than Bran

### `weapon_overlay`

Keep:

- bow only

Bow is required for this lane's combat-ready composite read.

### `upper_armor_overlay`

Keep:

- swappable upper hunter gear only

Must not carry:

- Tia's core asymmetry identity
- giant shoulder mass
- anything that turns her into a heavy fighter

### `shield_overlay`

Optional and likely unused.

If created later, it must not become a default part of the lane read.
