# Sprite Anchor Tia Prompt Pack V01

## Purpose

This prompt pack generates the first production-facing concept sheets for `Sprite Anchor Tia`.
The goal is to lock silhouette, ranged-class identity, and restrained agile motion grammar for the 2D sprite lane.

Use this with:

- `/Volumes/AI/tactics/assets/characters/sprite_anchor_tia/spec.md`
- `/Volumes/AI/tactics/docs/character_sprite_pipeline.md`
- `/Volumes/AI/tactics/docs/character_sprite_style_lock_v01.md`

## Shared Rules

Apply these rules to every prompt in this pack:

- classic tactical JRPG sprite-sheet feel
- chibi-adjacent tactical proportion
- clean line discipline
- restrained cel-style shading
- gameplay-scale readability first
- transparent or plain neutral background
- no text, no numbering, no speech bubbles
- no cinematic composition
- no glossy modern anime finish
- projectile and motion effects stay secondary to the body and bow

## 1. Idle Sheet

```text
Use case: stylized-concept
Asset type: tactical RPG character sprite sheet
Primary request: idle animation sheet for a wary forest ranger named Tia, designed for a classic tactical JRPG battle sprite pipeline
Subject: compact female ranged hunter with slightly enlarged head, guarded expression, short or medium bow, asymmetrical hood or shoulder break, muted forest palette
Style/medium: classic tactical JRPG sprite-sheet concept art, clean line discipline, restrained cel-style shading, soft painterly warmth without glossy anime polish
Composition/framing: full-body gameplay sprite sheet on a plain neutral background, front-facing battle orientation, consistent frame-to-frame body scale, 8-frame idle loop concept
Lighting/mood: soft neutral lighting, readable silhouette, no dramatic rim light
Color palette: muted forest green, ash brown, dark leather, subdued cloth neutrals
Materials/textures: practical cloth, leather straps, wood bow, light trim armor
Constraints: ranged hunter class must read before personality; body silhouette must remain readable without effects; bow curve and asymmetry must stay clear; grounded cautious stance
Avoid: giant cape motion, long cinematic hair, glossy anime hunter finish, heavy armor bulk, text, panel numbers, background scenery
```
