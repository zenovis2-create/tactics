# Enemy Raider 8Dir Production Brief V01

## Goal

Produce the first eight-direction gameplay base for Enemy Raider.

Required views:

- `front`
- `front_right`
- `right`
- `back_right`
- `back`
- `back_left`
- `left`
- `front_left`


## Read Goal

Enemy Raider must still read as:

- hostile melee pressure first
- rigid authority infantry second

The main risk is collapse into:

- ally swordsman recolor or generic bandit

## Must Keep Across All Directions

- compressed hostile posture
- rigid upper-body angle
- restrained ember-red hostile cue

## Quality Bar

- front: clearest primary class read
- diagonals: true in-between orientation, not duplicated side views
- back: the class silhouette still reads from rear mass and posture
- left/right: melee threat remains readable in profile
- all eight: identity stays stable without relying on FX or background help

## Notes
