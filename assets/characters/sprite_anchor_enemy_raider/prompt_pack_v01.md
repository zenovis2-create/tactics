# Sprite Anchor Enemy Raider Prompt Pack V01

## Purpose

This prompt pack generates the first production-facing concept sheets for `Sprite Anchor Enemy Raider`.
The goal is to lock the baseline hostile melee infantry lane for the project.

Use this with:

- `/Volumes/AI/tactics/assets/characters/sprite_anchor_enemy_raider/spec.md`
- `/Volumes/AI/tactics/docs/character_sprite_pipeline.md`
- `/Volumes/AI/tactics/docs/character_class_silhouette_color_matrix_v01.md`
- `/Volumes/AI/tactics/docs/production/character_visual_identity_pack.md`

## Shared Rules

Apply these rules to every prompt in this pack:

- classic tactical JRPG sprite-sheet feel
- chibi-adjacent tactical proportion
- clean line discipline
- restrained cel-style shading
- gameplay-scale readability first
- transparent or plain neutral background
- no text, no numbering, no speech bubbles
- no cinematic composition
- no western gritty realism
- no giant FX

## 1. Idle Sheet

```text
Use case: stylized-concept
Asset type: tactical RPG enemy sprite sheet
Primary request: idle animation sheet for a hostile empire pursuit soldier, designed for a classic tactical JRPG battle sprite pipeline
Subject: compact enemy melee infantry with slightly enlarged head, rigid posture, short sword, harsher authority silhouette, ember-red hostile accent, cold iron and leather gear
Style/medium: classic tactical JRPG sprite-sheet concept art, clean line discipline, restrained cel-style shading, same rendering family as allied sprites
Composition/framing: full-body gameplay sprite sheet on a plain neutral background, front-facing battle orientation, consistent frame-to-frame body scale, 8-frame idle loop concept
Lighting/mood: soft neutral production lighting, no dramatic background
Color palette: cold iron gray, soot leather, dark authority cloth, restrained ember-red accent
Constraints: unit must read as hostile before color; posture should feel compressed and disciplined; silhouette must remain distinct from ally swordsmen
Avoid: ally warmth, gritty western realism, bandit sloppiness, monstrous distortion, magical aura, text, panel numbers, background scenery
```
