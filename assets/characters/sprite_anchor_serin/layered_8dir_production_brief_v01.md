# Serin Layered 8Dir Production Brief V01

## Goal

Migrate Serin from monolithic eight-direction prep into a layered 8-direction
support/caster lane.

This lane is not meant to prove heavy armor or shield logic.

It is meant to prove:

- support/healer first read
- caster second read
- staff-required layer separation

## Required Layers

- `base_body`
- `base_outfit`
- `weapon_overlay`

## Optional Layers

- `shield_overlay`
- `upper_armor_overlay`

## Direction Set

- `front`
- `front_right`
- `right`
- `back_right`
- `back`
- `back_left`
- `left`
- `front_left`

## Read Goal

Serin must still read as:

- support first
- healer first
- caster second
- calm tactical presence

The main risks are:

- collapsing into a generic child mage
- letting upper gear replace the robe/support identity
- losing staff clarity in profile and diagonal views

## Layer Guidance

### `base_body`

Keep:

- head
- face
- hair
- body mass
- posture and arm structure

Must not carry:

- staff
- shield
- swappable upper gear mass

### `base_outfit`

Keep:

- robe identity
- lower-body cloth
- support/healer silhouette
- stable non-swappable support garment read

Must not become:

- full armor shell
- generic bulky caster costume

### `weapon_overlay`

Keep:

- staff only

This layer is required.

### `shield_overlay`

Optional.

Likely unused for Serin.

### `upper_armor_overlay`

Only use for:

- swappable upper support gear
- light shoulder or chest support equipment

Must not carry:

- the whole robe identity
- the entire support read

## Legacy Rule

Any previous monolithic eight-direction prep remains:

- reference-only
- not final layered production truth
