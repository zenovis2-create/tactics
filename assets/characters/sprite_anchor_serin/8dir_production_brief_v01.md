# Serin 8Dir Production Brief V01

## Goal

Produce the first eight-direction gameplay base for Serin.

Required views:

- `front`
- `front_right`
- `right`
- `back_right`
- `back`
- `back_left`
- `left`
- `front_left`


## Read Goal

Serin must still read as:

- support first
- caster second

The main risk is collapse into:

- generic child mage or robe blob without class signal

## Must Keep Across All Directions

- compact staff read
- soft robe mass with clear outer contour
- calm posture

## Quality Bar

- front: clearest primary class read
- diagonals: true in-between orientation, not duplicated side views
- back: the class silhouette still reads from rear mass and posture
- left/right: staff and support posture remain visible in profile
- all eight: identity stays stable without relying on FX or background help
