# Rian 8Dir Production Brief V01

## Goal

Produce the first eight-direction gameplay base for Rian.

Required views:

- `front`
- `front_right`
- `right`
- `back_right`
- `back`
- `back_left`
- `left`
- `front_left`


## Read Goal

Rian must still read as:

- frontline command
- light sword leadership

The main risk is collapse into:

- generic swordsman or Bran-like heavy silhouette

## Must Keep Across All Directions

- mantle or cloak split
- compact sword-arm read
- stable shoulder line

- front: clearest primary class read
