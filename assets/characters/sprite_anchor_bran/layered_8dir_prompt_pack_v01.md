# Sprite Anchor Bran Layered 8Dir Prompt Pack V01

## Purpose

This prompt pack generates the first layered 8-direction production set for
`Sprite Anchor Bran`.

The goal is to prove a heavy defender can survive visible equipment separation,
especially the shield layer.

## Use With

- `/Volumes/AI/tactics/assets/characters/sprite_anchor_bran/spec.md`
- `/Volumes/AI/tactics/assets/characters/sprite_anchor_bran/layered_8dir_production_brief_v01.md`

- `/Volumes/AI/tactics/docs/style_bible.md`

## Shared Rules

Apply these rules to every prompt in this pack:

- classic tactical JRPG sprite-style character asset
- single-layer directional concept only
- 8-direction coverage required
- clean line discipline
- restrained cel-style shading
- gameplay-scale readability first
- plain neutral or transparent background
- no text, no numbering, no labels
- no cinematic composition
- no oversized FX
- preserve shield-first heavy-defender readability in every orientation

## Layer 1: Base Body

```text
Use case: stylized-concept
Asset type: layered tactical RPG 8-direction character base
Primary request: generate Bran's base body layer across front, front-right, right, back-right, back, back-left, left, and front-left views
Subject: veteran heavy defender body mass with broad shoulders, planted stance, readable head and face, no shield, no swappable upper armor shell
Style/medium: classic tactical JRPG character base art, clean line discipline, restrained cel-style shading
Composition/framing: single-direction views intended for an 8-view layered turnaround set, full-body, centered, neutral stance, plain neutral background
Lighting/mood: soft neutral lighting
Constraints: preserve heavy defender first and shield-wall veteran second through body mass alone; do not let the body become generic knight noise; no gear layer baked in
Avoid: shield silhouette baked into body, giant armor shell, text, numbering, scenery, extra props
```
