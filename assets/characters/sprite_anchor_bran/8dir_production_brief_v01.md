# Bran 8Dir Production Brief V01

## Goal

Produce the first eight-direction gameplay base for Bran.

Required views:

- `front`
- `front_right`
- `right`
- `back_right`
- `back`
- `back_left`
- `left`
- `front_left`


## Read Goal

Bran must still read as:

- heavy defender first
- shield-wall veteran second

The main risk is collapse into:

- generic knight or Rian with more armor

## Must Keep Across All Directions

- large shield mass
- broad shoulders
- compact heavy stance

- front: clearest primary class read
