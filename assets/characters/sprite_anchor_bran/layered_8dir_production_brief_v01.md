# Bran Layered 8Dir Production Brief V01

## Goal

Migrate Bran from a monolithic directional sheet into a layered 8-direction
character stack.

## Required Layers

- `base_body`
- `base_outfit`
- `weapon_overlay`
- `shield_overlay`
- `upper_armor_overlay`

## Direction Set

- `front`
- `front_right`
- `right`
- `back_right`
- `back`
- `back_left`
- `left`
- `front_left`

## Read Goal

Bran must still read as:

- heavy defender first
- shield-wall veteran second
- fortress-like endurance over speed

The main risk is:

- losing the shield-first class identity once the body is separated from gear

## Layer Guidance

### `base_body`

Keep:

- head
- face
- broad body mass
- planted stance

Must not carry:

- shield
- upper armor shell

### `base_outfit`

Keep:

- heavy under-cloth silhouette
- lower-body mass
- non-swappable cloth base

### `weapon_overlay`

Keep:

- short sword only

Weapon is secondary to shield read.

### `shield_overlay`

Keep:

- shield only

This is the key proof layer for the pilot's heavy variant logic.

### `upper_armor_overlay`

Keep:

- chest armor
- shoulder armor
- upper defensive mass that should visibly change with gear

## Heavy Proof Rule

If the shield overlay does not remain the dominant class-defining read, the
layer split fails.
