# Sprite Anchor Bran Prompt Pack V01

## Purpose

This prompt pack generates the first production-facing concept sheets for `Sprite Anchor Bran`.
The goal is to lock silhouette, heavy-defender identity, and grounded wall-line motion grammar for the 2D sprite lane.

Use this with:

- `/Volumes/AI/tactics/assets/characters/sprite_anchor_bran/spec.md`
- `/Volumes/AI/tactics/docs/character_sprite_pipeline.md`
- `/Volumes/AI/tactics/docs/character_class_silhouette_color_matrix_v01.md`

## Shared Rules

Apply these rules to every prompt in this pack:

- classic tactical JRPG sprite-sheet feel
- chibi-adjacent tactical proportion
- clean line discipline
- restrained cel-style shading
- gameplay-scale readability first
- transparent or plain neutral background
- no text, no numbering, no speech bubbles
- no cinematic composition
- no glossy modern anime finish
- no FX-led identity

## 1. Idle Sheet

```text
Use case: stylized-concept
Asset type: tactical RPG character sprite sheet
Primary request: idle animation sheet for a heavy wall-defender knight named Bran, designed for a classic tactical JRPG battle sprite pipeline
Subject: veteran heavy knight with compact chibi-adjacent tactical proportions, broad shoulders, large shield, short sword, scarred stern face, restrained dark cloth accent
Style/medium: classic tactical JRPG sprite-sheet concept art, clean line discipline, restrained cel-style shading, soft painterly warmth without glossy anime polish
Composition/framing: full-body gameplay sprite sheet on a plain neutral background, front-facing battle orientation, consistent frame-to-frame body scale, 8-frame idle loop concept
Lighting/mood: soft neutral lighting, readable silhouette, no dramatic rim light
Color palette: cold steel gray, worn leather brown, muted fortress blue cloth accent
Materials/textures: steel armor, leather straps, heavy cloth, large shield
Constraints: heavy defender class must read before personality; shield must remain the dominant gear read; silhouette must be clearly heavier than a standard swordsman; grounded stance
Avoid: giant cape drama, luxury gold knight trim, giant fantasy weapon, magical aura, flashy hero posing, text, panel numbers, background scenery
```
