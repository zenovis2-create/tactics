# Forest Tile 02 Spec

## Identity

- Asset ID: `tile_forest_02`
- Working name: `Forest Tile 02`
- Tile family: forest
- Chapter or biome: Greenwood / general woodland support set

## Purpose

- Primary use: second baseline forest terrain tile to reduce repetition in tactical boards
- Runtime slot: tile card, tile icon, board reference
- Gameplay role: soft cover, movement tax, natural route contrast

## Visual Summary

- One-sentence pitch: a forest tile variant with denser side brush and offset root rhythm that still keeps a readable traversable center
- First-read terrain cue: darker edge clusters with a cleaner center lane than Forest Tile 01
- Gameplay meaning: slower movement, partial protection, alternate woodland board rhythm

## Shape Rules

- Keep the center traversable
- Push density to edges and one corner cluster
- Use different root flow from Forest Tile 01
- Avoid full-tile bush mass or vertical obstruction

## Color Plan

- Base palette: muted moss green, dark soil brown, cool shadow green
- Accent color: subdued pale green edge highlight
- Contrast requirement: distinct from plain tile, but character-safe

## Runtime Goal

- Must sit beside `forest_tile_01` without feeling like a recolor only
- Must remain readable under units and UI overlays
