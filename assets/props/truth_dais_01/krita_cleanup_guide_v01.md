# Truth Dais 01 Krita Cleanup Guide V01

## Purpose

This guide converts the current truth-dais image into a reusable runtime-facing chapter prop.

## Expected Source

When a chosen truth-dais candidate is placed in:

- `source/`

use this guide to produce the clean and runtime outputs.

## Cleanup Goal

Produce a clean archive landmark asset that:

- has transparent background
- preserves dais / top plane / brace separation
- stays evidence-first and altar-second
- can support chapter map previews, landmark cards, and icon surfaces

## Step 1. Remove Background

- keep full transparency
- remove all paper, glow, and framing
- do not keep drop-shadow haze beyond a minimal contact-softening edge

## Step 2. Preserve Major Structure

Keep:

- readable dais mass
- visible top plane
- clear support braces
- one restrained accent only

Do not add:

- giant codex piles
- decorative quill clutter
- crystal spectacle

## Step 3. Tighten Value Separation

- top plane must stay distinct from the base
- brace logic must read before the accent
- accent must stay tertiary

## Step 4. Prepare Reusable Output

The cleaned dais should support:

- chapter landmark sheet
- map integration surface
- future icon extraction

That means the silhouette must remain clean and centered.

## Step 5. Export Clean Output

Export:

- `truth_dais_01_clean_v01.png`

Store it under:

- `/Volumes/AI/tactics/assets/props/truth_dais_01/clean/`

## Step 6. Prepare Runtime Derivatives

Current derivatives:

- `truth_dais_01_landmark_v01.png`
- `truth_dais_01_icon_v01.png`
- `truth_dais_01_integration_v01.png`

Store these under:

- `/Volumes/AI/tactics/assets/props/truth_dais_01/runtime/`

## Acceptance Checklist

- background fully removed
- dais, top plane, and braces remain distinct
- silhouette survives 128px reduction
- reads as truth dais, not altar or desk clutter
- sits in the same world as CH05 archive pressure and record landmarks
