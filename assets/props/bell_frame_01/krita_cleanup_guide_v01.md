# Bell Frame 01 Krita Cleanup Guide V01

## Purpose

This guide converts the current bell-frame image into a reusable runtime-facing chapter prop.

## Expected Source

When a chosen bell-frame candidate is placed in:

- `source/`

use this guide to produce the clean and runtime outputs.

## Cleanup Goal

Produce a clean ritual-city landmark asset that:

- has transparent background
- preserves frame / bell / accent separation
- stays civic-pressure-first and altar-second
- can support chapter map previews, landmark cards, and icon surfaces

## Step 1. Remove Background

- keep full transparency
- remove all paper, glow, and framing
- do not keep drop-shadow haze beyond a minimal contact-softening edge

## Step 2. Preserve Major Structure

Keep:

- readable bell mass
- visible support frame
- clear hanging point
- one restrained accent only

Do not add:

- giant bell arrays
- chain clutter
- decorative shrine overflow

## Step 3. Tighten Value Separation

- bell must stay distinct from the frame
- frame must read before the accent
- accent must stay tertiary

## Step 4. Prepare Reusable Output

The cleaned bell frame should support:

- chapter landmark sheet
- map integration surface
- future icon extraction

That means the silhouette must remain clean and centered.

## Step 5. Export Clean Output

Export:

- `bell_frame_01_clean_v01.png`

Store it under:

- `/Volumes/AI/tactics/assets/props/bell_frame_01/clean/`

## Step 6. Prepare Runtime Derivatives

Current derivatives:

- `bell_frame_01_landmark_v01.png`
- `bell_frame_01_icon_v01.png`
- `bell_frame_01_integration_v01.png`

Store these under:

- `/Volumes/AI/tactics/assets/props/bell_frame_01/runtime/`

## Acceptance Checklist

- background fully removed
- frame, bell, and accent remain distinct
- silhouette survives 128px reduction
- reads as bell frame, not shrine clutter or generic machinery
- sits in the same world as CH07 civic ritual and city-pressure landmarks
