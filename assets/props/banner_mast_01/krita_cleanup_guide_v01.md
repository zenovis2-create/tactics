# Banner Mast 01 Krita Cleanup Guide V01

## Purpose

This guide converts the current banner-mast image into a reusable runtime-facing chapter prop.

## Expected Source

When a chosen banner-mast candidate is placed in:

- `source/`

use this guide to produce the clean and runtime outputs.

## Cleanup Goal

Produce a clean fortress landmark asset that:

- has transparent background
- preserves mast / base / banner-plane separation
- stays identity-first and decoration-second
- can support chapter map previews, landmark cards, and icon surfaces

## Step 1. Remove Background

- keep full transparency
- remove all paper, glow, and framing
- do not keep drop-shadow haze beyond a minimal contact-softening edge

## Step 2. Preserve Major Structure

Keep:

- readable vertical mast
- visible banner plane
- clear base mass
- one restrained accent only

Do not add:

- parade drapery
- extra streamers
- giant crest clusters

## Step 3. Tighten Value Separation

- mast must stay distinct from the banner
- base must read before the accent cloth
- accent must stay tertiary

## Step 4. Prepare Reusable Output

The cleaned banner mast should support:

- chapter landmark sheet
- map integration surface
- future icon extraction

That means the silhouette must remain clean and centered.

## Step 5. Export Clean Output

Export:

- `banner_mast_01_clean_v01.png`

Store it under:

- `/Volumes/AI/tactics/assets/props/banner_mast_01/clean/`

## Step 6. Prepare Runtime Derivatives

Current derivatives:

- `banner_mast_01_landmark_v01.png`
- `banner_mast_01_icon_v01.png`
- `banner_mast_01_integration_v01.png`

Store these under:

- `/Volumes/AI/tactics/assets/props/banner_mast_01/runtime/`

## Acceptance Checklist

- background fully removed
- mast, banner plane, and base remain distinct
- silhouette survives 128px reduction
- reads as banner mast, not decorative pole clutter
- sits in the same world as fortress tiles and CH02 military landmarks
