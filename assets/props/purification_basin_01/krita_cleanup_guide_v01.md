# Purification Basin 01 Krita Cleanup Guide V01

## Purpose

This guide converts the current purification-basin image into a reusable runtime-facing chapter prop.

## Expected Source

When a chosen purification-basin candidate is placed in:

- `source/`

use this guide to produce the clean and runtime outputs.

## Cleanup Goal

Produce a clean sacred-machinery landmark asset that:

- has transparent background
- preserves basin / plane / base separation
- stays purification-first and altar-second
- can support chapter map previews, landmark cards, and icon surfaces

## Step 1. Remove Background

- keep full transparency
- remove all paper, glow, and framing
- do not keep drop-shadow haze beyond a minimal contact-softening edge

## Step 2. Preserve Major Structure

Keep:

- readable basin rim
- visible top plane
- clear base mass
- one restrained accent only

Do not add:

- giant fountain sprays
- statuary clutter
- crystal spectacle

## Step 3. Tighten Value Separation

- basin must stay distinct from the base
- top plane must read before the accent
- accent must stay tertiary

## Step 4. Prepare Reusable Output

The cleaned basin should support:

- chapter landmark sheet
- map integration surface
- future icon extraction

That means the silhouette must remain clean and centered.

## Step 5. Export Clean Output

Export:

- `purification_basin_01_clean_v01.png`

Store it under:

- `/Volumes/AI/tactics/assets/props/purification_basin_01/clean/`

## Step 6. Prepare Runtime Derivatives

Current derivatives:

- `purification_basin_01_integration_v01.png`

## Acceptance Checklist

- background fully removed
- basin, plane, and base remain distinct
- silhouette survives 128px reduction
- reads as purification basin, not altar or decorative fountain
- sits in the same world as CH04 sacred-machinery and monastery surfaces
