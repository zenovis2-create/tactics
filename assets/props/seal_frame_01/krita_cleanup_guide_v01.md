# Seal Frame 01 Krita Cleanup Guide V01

## Purpose

This guide converts the current seal-frame image into a reusable runtime-facing chapter prop.

## Expected Source

When a chosen seal-frame candidate is placed in:

- `source/`

use this guide to produce the clean and runtime outputs.

## Cleanup Goal

Produce a clean archive landmark asset that:

- has transparent background
- preserves frame / center / base separation
- stays seal-pressure-first and generic-route-device-second
- can support chapter map previews, landmark cards, and icon surfaces

## Step 1. Remove Background

- keep full transparency
- remove all paper, glow, and framing
- do not keep drop-shadow haze beyond a minimal contact-softening edge

## Step 2. Preserve Major Structure

Keep:

- readable outer frame
- visible sealed center
- clear support base
- one restrained accent only

Do not add:

- door panels
- giant arch detail
- decorative clutter

## Step 3. Tighten Value Separation

- frame must stay distinct from the center
- center must read before the accent
- accent must stay tertiary

## Step 4. Prepare Reusable Output

The cleaned seal frame should support:

- chapter landmark sheet
- map integration surface
- future icon extraction

That means the silhouette must remain clean and centered.

## Step 5. Export Clean Output

Export:

- `seal_frame_01_clean_v01.png`

Store it under:

- `/Volumes/AI/tactics/assets/props/seal_frame_01/clean/`

## Step 6. Prepare Runtime Derivatives

Current derivatives:

- `seal_frame_01_landmark_v01.png`
- `seal_frame_01_icon_v01.png`
- `seal_frame_01_integration_v01.png`

Store these under:

- `/Volumes/AI/tactics/assets/props/seal_frame_01/runtime/`

## Acceptance Checklist

- background fully removed
- frame, center, and base remain distinct
- silhouette survives 128px reduction
- reads as seal frame, not gate-control recolor or furniture
- sits in the same world as CH05 archive pressure and truth landmarks
