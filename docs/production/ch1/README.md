# Chapter 1 Production Pack

이 패키지는 `1장. 이름 없는 새벽`을 구현 가능한 형태로 고정하기 위한 실무용 묶음 문서다.

## 정본 우선순위

1. 장 구조/번호/기억 조각: `camp.md`
2. 1장 구현 상세(맵, 조건, 트리거, 대사): `phase1.md`
3. 상위 테마/후반 장 연결: `synopsis.md`
4. 충돌 판정 규칙: `docs/story_alignment.md`

## 포함 문서

- `ch1_narrative_lock.md`
- `ch1_dialogue_intro_outro_pass.md`
- `ch1_map_emotional_purpose.md`
- `ch1_encounter_pacing_variety_spec.md`
- `canon_terms_naming_sheet.md`
- `ch1_handoff_gameplay_ui_art_marketing.md`

## 사용 규칙

- 1장 관련 신규 기획, 아트, UI 카피, 마케팅 문구는 본 패키지 기준으로 검수한다.
- 본 패키지와 `phase1.md`가 충돌하면, 구현 데이터(ID/좌표/승패 조건)는 `phase1.md`를 우선한다.
- 본 패키지와 `camp.md`가 충돌하면, 장 구조/핵심 기억 조각/주제는 `camp.md`를 우선한다.

## 캐논 게이트 체크리스트

- 리안은 “기억상실 피해자”가 아니라 “전술 설계의 과거를 가진 인물”로만 묘사된다.
- 세린은 “수동 힐러”가 아니라 “신념 기반의 보호자” 목소리를 유지한다.
- 1장 감정선은 `각성 -> 신뢰의 씨앗 -> 동행 서약` 순서를 벗어나지 않는다.
- 핵심 질문인 `기억을 지워 평화를 얻을 것인가, 기억한 채 앞으로 갈 것인가`를 흐리는 표현을 금지한다.
- 마케팅/아트에서 `제로`, `무명의 종`의 정체를 1장 범위 밖에서 선공개하지 않는다.
