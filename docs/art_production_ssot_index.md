# Art Production SSOT Index

## Purpose

This is the source-of-truth index for the current Farland Tactics art production lane.

Use this document to answer three questions quickly:

1. what visual rules are locked
2. which asset templates should be used
3. where the current production-ready anchor assets live

This document is an index.
It does not replace the detailed source documents it points to.

## Top-Level Rule

The current art lane is split into two major surfaces:

- `character sprites`
- `environment / props / tiles`

Both must share the same world and readability standards even when the toolchains differ.

## Recommended Workflow

### New Character

1. pick the closest class lane from the character matrix
2. copy the sprite character template
3. write the asset spec
4. generate concept sheets
5. lock chosen source versions
6. clean and slice
7. review in Godot

### New Prop / Tile / Map

1. pick the closest environment lane from the environment matrix
2. copy the relevant template
3. write the asset spec
4. generate concept support if needed
5. block out in Rhino
6. paint over or extract in Krita
7. review in Godot
