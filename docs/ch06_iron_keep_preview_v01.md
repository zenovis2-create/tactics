# CH06 Iron Keep Preview V01

## Purpose

This preview validates whether the current fortified-ground and interaction families still hold when the scene shifts from early fortress pressure into heavier iron-keep / siege territory.

It asks:

`Can the current fortress family survive a harsher, more military CH06-like presentation without collapsing into generic dark noise?`

## Required Ingredients

The frame should include:

- Rian
- Bran
- Enemy Raider
- Enemy Skirmisher
- fortress_tile_01
- fortress_tile_02
- altar_01
- lever_01
- gate_control_01
- paladin_shield

## Review Focus

Compared to the CH02 pass, this preview should push:

- heavier siege pressure
- stronger military geometry
- more severe ground tone
- clearer route/system-control reading

## Intended Read Order

1. frontline clash
2. enemy pressure
3. route-control machinery
4. sacred anchor
5. fortified ground family
6. support equipment surface

If the floor or machinery becomes the main visual read before the units, the preview fails.

## Pass Conditions

- the fortress family still reads as engineered ground, not visual clutter
- Bran remains heavier than Rian even under darker fortified values
- Enemy Raider and Enemy Skirmisher still separate from each other and from the allies
- altar still reads sacred
- lever still reads local mechanism
- gate control still reads route/system machinery

## Working Goal

This preview is the second major proof point after CH02.

If it passes, the fortress family can be treated as robust across both:

- early fortress spaces
- late fortified / siege spaces
