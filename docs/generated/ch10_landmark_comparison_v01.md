## Comparison Goal

The player should be able to read these objects before UI help as:

- anchor chain: bell-lane suppression and terminal route reopening
- bell dais: final toll pressure and terminal ritual focus
- altar: sacred anchor or witness object
- lever: local mechanism
- gate control: broader route or system control

If CH10 landmarks collapse into altar or gate-control language, chapter-local clarity has failed.

## Current Read Check

### Altar

### Lever

- First read: local hand-operated mechanism

### Gate Control

- First read: broader route or system control

## Current Conclusion

CH10 now has a usable chapter-local landmark split:

- `anchor_chain_01` for bell-lane suppression and terminal route reopening
- `bell_dais_01` for final toll pressure and terminal ritual focus

This allows altar, lever, and gate control to keep their broader meanings instead of doing all endgame interaction work alone.
