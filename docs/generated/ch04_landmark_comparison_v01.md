## Comparison Goal

The player should be able to read these objects before UI help as:

- floodgate wheel: water-control and route-state landmark
- purification basin: purification target and safe activation landmark
- altar: sacred objective
- lever: local hand-operated mechanism
- gate control: larger route or system control

If CH04 landmarks collapse into altar or gate-control language, chapter-local clarity has failed.
