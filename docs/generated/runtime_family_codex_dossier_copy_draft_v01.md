## Output Format

Each family gets:

1. codex card title
2. category line
3. short paragraph
4. dossier comparison note
5. reference strip label

The goal is stable object identity, not battle instruction.

## Writing Guardrails

Apply these rules if the copy is revised later:

1. title must remain a short noun phrase
2. category line must stay one sentence
3. paragraph should explain category meaning, not battle usage
4. comparison note should clarify identity, not just shape
5. avoid tactical verbs like `disable`, `secure`, or `release`
