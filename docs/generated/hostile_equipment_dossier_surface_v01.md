# Hostile Equipment Dossier Surface V01

## Purpose

This document defines the first hostile-equipment dossier surface for the current slice.

It does not wire a new runtime scene by itself.
It prepares a stable hostile-gear presentation surface that can be promoted into:

- enemy dossier cards
- codex or records UI
- battle-preview support cards

## Included Hostile Equipment

- `hostile_field_blade_01`
- `hostile_short_bow_01`
- `hostile_shield_01`
- `hostile_armor_support_01`

## Surface Goal

The board should let a reviewer answer three questions in one glance:

1. Do hostile gear families look like one faction?
2. Are hostile melee and hostile ranged lanes still distinct?
3. Does hostile equipment still belong to the same world as ally equipment without collapsing into the same faction read?

## Recommended Immediate Uses

Use this board in:

- enemy dossier concept review
- hostile roster documentation
- codex or records surface planning

Do not yet treat it as a final in-game UI export.

## Promotion Rule

Promote this board or its derived crops only when:

- a real hostile dossier or codex slot exists
- the hostile lane is no longer expected to change its faction grammar in the near term
