## Comparison Goal

The player should be able to read these objects before UI help as:

- shield wreck: true cover and siege aftermath landmark
- chain-lift winch: navigation objective and siege engineering device
- altar: sacred anchor
- lever: local mechanism
- gate control: larger route or system control

If CH06 landmarks collapse into gate-control or generic clutter language, chapter-local clarity has failed.
