## Comparison Goal

The player should be able to read these objects before UI help as:

- truth dais: evidence-bearing truth anchor
- seal frame: unseal and revision-pressure device
- altar: sacred or truth-bearing objective
- lever: local mechanism
- gate control: larger route or system control

If CH05 landmarks collapse into altar or gate-control language, chapter-local clarity has failed.
