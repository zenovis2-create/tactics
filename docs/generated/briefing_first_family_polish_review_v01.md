# Briefing-First Family Polish Review V01

## Purpose

This document defines the review criteria for the current briefing-first family
set.

It is a polish document, not a new implementation spec.

The goal is to judge whether the authored briefing surfaces already read cleanly
enough for player-facing use.

## Scope

Families under review:

- `battery`
- `floodgate`
- `chain_control`

Stage anchors:

- `battery` -> `CH06_02`
- `floodgate` -> `CH04_03`
- `chain_control` -> `CH10_05`

## Review Targets

Each stage should be reviewed across these three surfaces:

1. summary lead sentence
2. enemy intel and terrain summary wording
3. optional-objective framing

## Family-Specific Review Standard

### `battery`

The briefing should read as:

- line pressure
- siege geometry
- exposed advance risk

It should not read as:

- generic fortress machinery
- neutral objective furniture

Fast review questions:

- does the stage read like a battery line, not just a keep interior
- does the text make cover and lane pressure feel central

### `floodgate`

The briefing should read as:

- route-state control
- water-state change
- crossing safety shift

It should not read as:

- generic monastery lore
- generic reliquary approach text

Fast review questions:

- does the player understand that the route itself changes
- does the text foreground control of water rather than just place flavor

### `chain_control`

The briefing should read as:

- terminal release
- final approach control
- endgame lane relief

It should not read as:

- generic boss-fight hype
- general tower atmosphere without a control read

Fast review questions:

- does the player understand that the anchor chain changes the final approach
- does the copy preserve an endgame-control read rather than a background prop read

## Cross-Family Review Questions

- does each family still read differently at briefing scale
- is any stage drifting back into chapter flavor instead of tactical read
- are objective and terrain lines reinforcing the same family read rather than fighting each other

## Quality Bar

The authored briefing passes polish review if:

1. the family read is visible in the first paragraph
2. the terrain summary reinforces that same read
3. the optional objectives do not distract from the core family read
4. the stage still feels specific rather than boilerplate

## Working Conclusion

The briefing-first polish pass should focus on whether the current authored text
preserves family identity under compressed panel reading.
