# Manual Play Feedback Template V01

## How To Use

For each issue you notice during manual play, record one block.

Keep each block short and concrete.

## Template

```text
장면:
항목:
문제:
```

## Valid `항목` Values

- `Unit Readability`
- `Movement Feel`
- `Attack Timing`
- `Hit / Defeat Feel`
- `Board Surface`
- `Chapter Identity`
- `HUD / Framing`

## Good Examples

```text
장면: CH07 representative battle
항목: Chapter Identity
문제: city seal보다 prayer dais가 먼저 안 보임
```

```text
장면: CH09B representative battle
항목: HUD / Framing
문제: 하단 패널이 archive lectern 근처 전투 공간을 너무 많이 가림
```

```text
장면: CH10 representative battle
항목: Attack Timing
문제: support와 ranged가 여전히 비슷하게 느껴짐
```

## Review Order

1. `CH07` representative battle
2. `CH09B` representative battle
3. `CH10` representative battle
4. ranged/support가 자주 나오는 일반 battle
5. movement/melee 교환이 많은 일반 battle

## Rule

Do not bundle multiple unrelated issues into one block.

One issue per block is faster to fix and easier to verify.
