## Comparison Goal

The player should be able to read these objects before UI help as:

- split marker post: divided-route guidance and commitment pressure
- transfer gate latch: route release and blocked-lane relief
- altar: pursuit mark / memory anchor / ruin shrine
- lever: local release or hunter-device style trigger
- gate control: engineered remnant that changes route state

If CH08 landmarks collapse into altar or gate-control language, chapter-local clarity has failed.
