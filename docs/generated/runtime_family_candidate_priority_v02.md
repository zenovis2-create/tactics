## Wave 2 Candidates

## Working Conclusion

The project should not keep adding runtime families by default.

The next family should be chosen by:

1. strongest silhouette at icon scale
2. strongest gameplay contract
3. lowest overlap with an existing live family

By that standard, `bell_frame_01` is the best next runtime family candidate.
