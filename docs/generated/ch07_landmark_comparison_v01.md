## Comparison Goal

The player should be able to read these objects before UI help as:

- bell frame: civic ritual pressure and public warning
- city seal dais: civic oath and city-seal objective anchor
- altar: prayer / memory / civic oath anchor
- lever: local release or hand-operated city mechanism
- gate control: broader route-control infrastructure

If CH07 landmarks collapse into altar or gate-control language, chapter-local clarity has failed.
